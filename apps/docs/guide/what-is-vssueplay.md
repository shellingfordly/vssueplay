# What is Vssueplay?

Vssueplay (/'vɪʃuːpleɪ/, meaning is vue-issue-replay) is a Vue3 component / plugin, which can enable comments for your static pages.

This project inspiration comes in [Vssue](https://github.com/meteorlxy/vssue). The Vssueplay is the Vue3 version of Vssue.
